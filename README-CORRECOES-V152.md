# One Dice Site v1.52 - restauração do layout e correções seguras

Base usada: v1.48 estável.

Esta versão não usa o rebuild simplificado da v1.51. O layout original foi preservado.

Correções aplicadas:
- Defesa e Esquiva independentes, com restauração do campo oposto após handlers legados.
- Texto auxiliar Manual oculto.
- Ficha aberta pela aba Personagens força modo ficha avulsa.
- Redução de flicker ao editar campos da ficha.
- Barra rápida oculta por padrão, com alternância no menu.
- Perfil da sessão clicável para abrir configurações da conta.
- Versão/cache atualizado para 1.52.0.

Observação: as camadas antigas não foram removidas em massa nesta versão para não perder layout e funções. A limpeza estrutural deve ser feita depois, arquivo por arquivo, com testes.
