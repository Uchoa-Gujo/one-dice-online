# Passo a passo iniciante — etapa 1

Nesta etapa você ainda não precisa comprar nada.

## O que esta versão faz

Ela prepara o projeto para rodar em servidor com Docker.

Você terá:

- site em `client/`
- servidor em `server/`
- banco em `database/schema.sql`
- configuração Docker em `docker-compose.yml`

## Próximo passo depois desta versão

1. Criar conta no GitHub.
2. Criar um repositório chamado `one-dice-online`.
3. Enviar esses arquivos para o GitHub.
4. Comprar VPS Hostinger quando estivermos prontos.
5. Subir o projeto na VPS.

## Observação

Você pediu para não rodar no PC. Tudo bem.
Esta versão foi montada para ser levada direto ao GitHub/VPS depois.
